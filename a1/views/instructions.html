Instructions for Participants

About

In the next screen, there will be a trial for you to familiarize yourself with the two techniques. The interface is split into two sections. In the left section, there will be a string of text that you are required to copy, while in the right section, you will be shown a few windows containing text, one of which is a Text Editor window which you can paste text in.

There are two ways you can copy and paste the text:

Using Keyboard Shortcuts

Firstly highlight the text that you want to copy, then press Cmd + C to copy the text into your clipboard. Next, click on the location where you want to paste the text, then press Cmd + V to paste the text.

Using AutoComPaste

AutoComPaste is a new copy and paste method which analyzes the text in your currently opened windows. Start typing the text you want to copy and you will see a dropdown list of possibilities. Use the up/down arrow keys to select the correct line that you want to copy and press Enter.



The Experiment

For the real experiment, there will be 27 trials per technique and you are given a minute's rest between each technique.

Press the proceed button when you are ready.
